#OVERVIEW
Aplicación de escritorio desarrollada en Objective-C con Cocoa que funciona como una calculadora científica interactiva con interfaz gráfica para macOS.
Permite realizar operaciones matemáticas básicas y avanzadas, incluyendo:
 + Suma, resta, multiplicación y división
 + Potencias y raíces cuadradas
 + Factorial (iterativo y recursivo)
 + Verificación de números primos
 + Conversión a números romanos
 + Cálculo de funciones trigonométricas (seno, coseno y tangente)

#FILES
