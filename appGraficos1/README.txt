#Overview
aplicación para macOS desarrollada en Objective-C utilizando el framework Cocoa.
El objetivo principal es practicar técnicas de dibujo en 2D mediante el uso de NSView, NSBezierPath, 
y NSGradient, implementando diversas figuras, curvas Bezier, gradientes y un dibujo personalizado complejo.
