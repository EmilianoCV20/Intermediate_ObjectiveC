#Overview
Aplicación de consola desarrollada en Objective-C, enfocada en practicar el uso de NSString, 
NSMutableString y sus métodos principales.
El programa demuestra cómo manipular cadenas mediante:
+ Concatenación
+ Búsqueda de subcadenas
+ Reemplazo de caracteres
+ Obtención de subcadenas
+ Eliminación de partes del texto
+ Inversión de una cadena usando enumeración de subcadenas
